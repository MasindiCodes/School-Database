# School-Database
A simple system used to understand an organization’s operations through research, while identifying problems that can be solved using a database as well as 
designing and implementing a complete relational database solution
## Description
This project focuses on the design and implementation of a relational database system to solve real-world problems faced by an organization.
It applies the following database design principles:
-Data modeling
-Normalization
-Database implementation using Microsoft SQL Server
-Writing T-SQL scripts
-Creating database objects such as tables, views, stored procedures, and triggers
